#+title: Docker commands for getting information about Containers

| Command                                                                      | Notes                                                                  |
|------------------------------------------------------------------------------+------------------------------------------------------------------------|
| ~docker ps~                                                                  | Running Containers                                                     |
| ~docker ps -a~                                                               | ~                                                                      |
| ~docker logs <image>~                                                        | Container logs                                                         |
| ~docket inspect <image>~                                                     | Inspecting Containers                                                  |
| ~docker inspect --format '{{ .NetworkSettings.IPAddress }}' $(docker ps -q)~ | ~                                                                      |
| ~docker events <image>~                                                      | Containers events                                                      |
| ~docker top <image>~                                                         | Running process                                                        |
| ~docker port <image>~                                                        | Public ports                                                           |
| ~docker stats <image>~                                                       | Container resource usage                                               |
| ~docker diff <image>~                                                        | Inspecting changes to files or directories on a Container's filesystem |
