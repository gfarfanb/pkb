#+title: Docker commands for manipulating Images

| Command                                                        | Notes                                                                |
|----------------------------------------------------------------+----------------------------------------------------------------------|
| ~docker images~                                                | Listing Images                                                       |
| ~docker build .~                                               | Building Images                                                      |
| ~docker build github.com/<user>/<repo>~                        | ~                                                                    |
| ~docker build - < Dockerfile~                                  | ~                                                                    |
| ~docker build - < <filename>.tar.gz~                           | ~                                                                    |
| ~docker build -t <image> .~                                    | ~                                                                    |
| ~docker build -f <other-docker-file> .~                        | ~                                                                    |
| ~curl example.com/remote/Dockerfile \vert docker build -f - .~ | ~                                                                    |
| ~docker rmi <image>~                                           | Removing an Image                                                    |
| ~docker load < ubuntu.tar.gz~                                  | Loading a tarred repository from a file or the Standard Input Stream |
| ~docker load --input ubuntu.tar~                               | ~                                                                    |
| ~docker save busybox > ubuntu.tar~                             | Save an Image tp a TAR archive                                       |
| ~docker history~                                               | Showing the history of an Image                                      |
| ~docker commit <image>~                                        | Creating an Image from a Container                                   |
| ~docker tag <tag> <image>~                                     | Tagging an Image                                                     |
| ~docker push <image>~                                          | Pushing an Image                                                     |
