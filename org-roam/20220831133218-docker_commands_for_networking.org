#+title: Docker commands for networking

| Command                                                                                                                                                                                                                                                                                                                              | Notes                                              |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+----------------------------------------------------|
| ~docker network create -d overlay MyOverlayNetwork~                                                                                                                                                                                                                                                                                  | Creating Networks                                  |
| ~docker network create -d bridge MyBridgeNetwork~                                                                                                                                                                                                                                                                                    | ~                                                  |
| ~docker network create -d overlay --subnet=192.168.0.0/16 --subnet=192.170.0.0/16 --gateway=192.168.0.100 --gateway=192.170.0.100 --ip-range=192.168.1.0/24 --aux-address="my-router=192.168.1.5" --aux-address="my-switch=192.168.1.6" --aux-address="my-printer=192.170.1.5" --aux-address="my-nas=192.170.1.6"  MyOverlayNetwork~ | ~                                                  |
| ~docker network rm MyOverlayNetwork~                                                                                                                                                                                                                                                                                                 | Removing a Network                                 |
| ~docker network ls~                                                                                                                                                                                                                                                                                                                  | Listing Networks                                   |
| ~docker network inspect MyOverlayNetwork~                                                                                                                                                                                                                                                                                            | Geeting information about a Network                |
| ~docker network connect MyOverlayNetwork <image>~                                                                                                                                                                                                                                                                                    | Connecting a running Container to a Network        |
| ~docker run -it -d --network=MyOverlayNetwork <image>~                                                                                                                                                                                                                                                                               | Connecting a Container to a Network when it starts |
| ~docker network disconnect MyOverlayNetwork <image>~                                                                                                                                                                                                                                                                                 | Disconnecting a Container from a Network           |
