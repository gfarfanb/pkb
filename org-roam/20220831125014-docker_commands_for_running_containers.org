#+title: Docker commands for running Containers

| Command                                           | Notes                |
|---------------------------------------------------+----------------------|
| ~docker create -t -i <image-name> --name <alias>~ | Creating a Container |
| ~docker run -it --name <alias> -d <image-name>~   | Running a Container  |
| ~docker rename <alias> <new-alias>~               | Renaming a Container |
| ~docker rm <alias>~                               | Removing a Container |
| ~docker update --cpu-shares 512 -m 300M <alias>~  | Updating a Container |
