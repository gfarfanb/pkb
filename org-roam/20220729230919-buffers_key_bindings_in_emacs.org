#+title: Buffers key bindings in Emacs
#+filetags: :emacs:command:

| Binding         | Command            | Notes                                             |
|-----------------+--------------------+---------------------------------------------------|
| *C-x C-b*       | /list-buffers/     | All active buffers                                |
| *C-x b*         | /switch-to-buffer/ |                                                   |
| *C-x k*         | /kill-buffer/      |                                                   |
| *C-x* /<RIGHT>/ | /next-buffer/      | Next buffer, according to active buffers list     |
| *C-x* /<LEFT>/  | /previous-buffer/  | Previous buffer, according to active buffers list |
| *C-x C-s*       | /save-buffer/      |                                                   |
