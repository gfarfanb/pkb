#+title: ref. Round-robin DNS
#+filetags: :refs:

Round-robin [[id:531d523f-fc03-4a7b-8b41-e934a76ee275][DNS]] is a load balancing technique where the balancing is done by a type of DNS server called an authoritative nameserver, rather than using a dedicated piece of load-balancing hardware.
