#+title: Docker commands for starting and stopping Containers

| Command                  | Notes                               |
|--------------------------+-------------------------------------|
| ~docker start <image>~   | Starting                            |
| ~docker stop <image>~    | Stopping                            |
| ~docker restart <image>~ | Restarting                          |
| ~docker pause <image>~   | Pausing                             |
| ~docker unpause <image>~ | Unpausing                           |
| ~docker wait <image>~    | Blocking a Container                |
| ~docker kill <image>~    | Sending a SIGKILL                   |
| ~docker attach <image>~  | Connecting to an existing Container |
