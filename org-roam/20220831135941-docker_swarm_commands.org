#+title: Docker Swarm commands

| Command                                                                                           | Notes                               |
|---------------------------------------------------------------------------------------------------+-------------------------------------|
| ~curl -ssl https://get.docker.com \vert bash~                                                     | Installing Docker Swarm             |
| ~docker swarm init --advertise-addr 192.168.10.1~                                                 | Initializing the Swarm              |
| ~docker swarm join-token worker~                                                                  | Getting a worker to join the Swarm  |
| ~docker swarm join-token manager~                                                                 | Getting a manager to join the Swarm |
| ~docker service ls~                                                                               | Listing Services                    |
| ~docker node ls~                                                                                  | Listing nodes                       |
| ~docker service create --name <alias> -p 8080:80 <image>~                                         | Creating a Service                  |
| ~docker service ps~                                                                               | Listing Swarm tasks                 |
| ~docker service scale <image>=3~                                                                  | Scaling a Service                   |
| ~docker service update --image <image>:<version> <alias>~                                         | Updating a Service                  |
