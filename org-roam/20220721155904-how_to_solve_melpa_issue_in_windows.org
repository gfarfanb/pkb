#+title: How to solve Melpa issue in Windows
#+filetags: :melpa:windows:emacs:

* Solution [/]
- [ ] Go to /package/ group:
  #+begin_src
    M-x customize-group RET package RET
  #+end_src
- [ ] Expand *Package Archives* then *INS*
- [ ] Fill in:
  | Field                 | Value                      |
  |-----------------------+----------------------------|
  | Archive name          | melpa                      |
  | URL or directory name | http://melpa.org/packages/ |
- [ ] Apply and Save
