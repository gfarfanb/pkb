#+title: Docker commands for Registries and Repositories

| Command                                              | Notes                 |
|------------------------------------------------------+-----------------------|
| ~docker login~                                       | Login to a Registry   |
| ~docker login localhost:8080~                        | ~                     |
| ~docker logout~                                      | Logout from a Regitry |
| ~docker logout localhost:8080~                       | ~                     |
| ~docker search <image>~                              | Searching an Image    |
| ~docker search <image> --stars=3 --no-trunc busybox~ | ~                     |
| ~docker pull <image>~                                | Pulling an Image      |
| ~docker pull <image> localhost:5000/myadmin/<image>~ | ~                     |
| ~docker push <image>~                                | Pushing an Image      |
| ~docker push <image> localhost:5000/myadmin/<image>~ | ~                     |
