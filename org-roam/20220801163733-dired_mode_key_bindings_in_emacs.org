#+title: Dired Mode key bindings in Emacs

| Ref     | Key               | Notes                                                              |
|---------+-------------------+--------------------------------------------------------------------|
| /<REF>/ | Return key        | Open files or directories in a new buffer                          |
| /(a)/   | (Lowercase) A key | (After activation) Open files or directorues in the current buffer |
| /(d)/   | (Lowercase) D key | Flagged for deletion                                               |
| /(u)/   | (Lowercase) U key | Remove deletion flag                                               |
| /(x)/   | (Lowercase) X key | Delete flagged entries                                             |
| /(C)/   | (Uppercase) C key | Copy                                                               |
| /(R)/   | (Uppercase) R key | Rename                                                             |
| /(+)/   | + key             | Create directory                                                   |
| /(!)/   | ! key             | Execute command on file at point                                   |
